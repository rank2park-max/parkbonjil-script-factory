# 한글 인코딩 테스트 가이드

Supabase UTF-8 인코딩 수정 후 아래 순서로 테스트하세요.

## 사전 준비
- Supabase URL과 Anon Key가 설정되어 있어야 합니다 (설정 페이지 또는 .env.local)
- `npm run dev`로 개발 서버 실행

## 테스트 단계

1. **홈페이지** (`/`) 접속
2. **영상 주제** 입력란에 `테스트 대본` 입력
3. **새 대본 시작** 버튼 클릭
4. 에디터 페이지(`/editor?id=xxx`)로 이동되는지 확인
5. **상단에 "테스트 대본" 제목이 정상 표시되는지** 확인
6. **내 프로젝트** (`/projects`)로 이동
7. 방금 만든 프로젝트의 제목이 **"테스트 대본"**으로 표시되는지 확인

## 기대 결과
- 모든 단계에서 한글이 `????` 없이 `테스트 대본`으로 정상 표시됨

## 추가 확인 (참고 자료)
- 설정 페이지의 **참고 자료** 섹션에서 한글 제목/내용이 있는 자료를 추가 후, 목록과 내용이 정상적으로 보이는지 확인
