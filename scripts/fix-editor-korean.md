# 에디터 한글 인코딩 수정 가이드

`src/app/editor/page.tsx` 파일이 UTF-8이 아닌 인코딩으로 저장되어 한글이 깨진 경우, 아래 방법으로 수정하세요.

## 방법 1: Cursor/VS Code에서 수동 수정

1. `src/app/editor/page.tsx` 파일 열기
2. **File > Save with Encoding** 선택 (또는 우측 하단 인코딩 클릭)
3. **UTF-8** 선택 후 저장
4. 아래 목록대로 검색(Ctrl+F)하여 깨진 문자열을 정상 한글로 교체:

| 검색 (깨진 문자열) | 교체 (정상 한글) |
|---|---|
| `????? ?? ? ????.` | `프로젝트를 찾을 수 없습니다.` |
| `???` (홈 버튼) | `홈으로` |
| `??? ??` (리서치 패널 헤더) | `리서치 패널` |
| `??` (목차) | `목차` |

**주의**: `data ?? null` 같은 JavaScript 연산자(`??`)는 절대 변경하지 마세요.

## 방법 2: .editorconfig 확인

프로젝트 루트에 `.editorconfig` 파일이 있으며, `charset = utf-8`로 설정되어 있습니다.  
편집기가 이 설정을 따르면 이후 저장 시 UTF-8로 저장됩니다.
